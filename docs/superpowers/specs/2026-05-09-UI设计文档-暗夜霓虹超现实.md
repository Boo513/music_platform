# UI 设计文档 — 暗夜霓虹超现实风格

## 1. 设计方向

| 属性 | 值 |
|------|-----|
| 风格 | 暗夜霓虹超现实（Dark Neon Surrealism） |
| 参考原型 | `preview/india-radio.html` |
| 核心视觉元素 | 深空背景 + 悬浮3D几何体 + 霓虹光束 + 玻璃态UI面板漂浮 |
| 色彩系统 | `--panel-bg: rgba(18,14,24,0.55)` `--accent: #FF8C42` `--accent-gold: #FFB366` `--text-primary: #f0e6e0` |
| 组件风格 | Glassmorphism: backdrop-blur + 半透明暗色背景 + 霓虹色边框 |

## 2. 页面清单

| 页面 | 路由 | 认证 | 说明 |
|------|------|------|------|
| 登录/注册 | `/login` `/register` | 否 | 同一页面，卡片翻转切换 |
| 首页 | `/` | 否 | 场景横幅 + 筛选 + 歌曲网格 |
| 上传音乐 | `/upload` | 是 | 拖拽 + 表单双栏 |
| 播放页 | `/play/:songId` | 否 | 全屏3D + 五区域UI覆盖层 |
| 个人主页 | `/profile` | 是 | 4Tab内容区 |

## 3. 页面详细设计

### 3.1 登录/注册页

**布局：** 全屏星河3D背景 + 玻璃态表单卡片悬浮中央 (380px宽)

**卡片翻转动画：** 登录/注册共享同一卡片，CSS transform: rotateY(180deg)，transition: 0.6s ease-in-out

**登录态：**
- 品牌标识 "SOUNDSCAPE" 左上角
- 卡片：🌌 图标 + "欢迎回来" + 用户名/密码输入框 + 霓虹渐变按钮 + "立即注册"链接

**注册态：**
- 卡片：✨ 图标 + "创建账号" + 用户名/昵称/密码输入框 + 霓虹渐变按钮 + "返回登录"链接

**交互：**
- 实时校验（失焦触发），错误时边框变红 + 提示文字
- 登录按钮 loading 态：旋转霓虹圆环
- 错误提示：卡片顶部滑入红色提示条，3秒自动消失
- 成功：卡片缩小消失 → 场景变亮 → 过渡到首页

### 3.2 首页

**6 个区域（从上到下）：**

| # | 区域 | 规格 |
|---|------|------|
| 1 | 场景预览横幅 | 180px高，左侧 Three.js Canvas 实时渲染当前场景 mini 版，右侧场景缩略图列表可点击切换。底部圆点指示器 |
| 2 | 搜索栏 + 快捷按钮 | 搜索框(圆角24px) + +上传按钮(40px圆形霓虹渐变) + 用户头像(36px) |
| 3 | 三个快捷卡片 | 推荐电台 / 热门推荐 / 最近播放 — 等宽玻璃态卡片，图标+标题+描述 |
| 4 | 风格筛选标签 | 横向滚动圆角标签。选中：霓虹橙色背景+边框；未选中：暗色玻璃态 |
| 5 | 歌曲网格 | 响应式网格。每卡：圆形封面(80px, 风格主题色光晕) + 歌名 + 艺术家 + 风格/情绪双标签。hover上浮+光晕增强 |
| 6 | 底部播放器栏 | 72px高，唱片旋转动画 + 霓虹进度条。与 india-radio 一致 |

**交互：**
- 场景横幅：点击左侧预览区 → 进入该场景对应推荐列表；点击右侧列表项 → 切换场景
- 歌曲卡片：点击 → 进入播放页 PlayPage，传递当前歌曲+当前队列
- +按钮 → 上传页
- 头像 → 个人主页
- 底部播放器 → 点击展开到全屏播放页

### 3.3 上传音乐页

**布局：** 左右双栏 — 左侧拖拽区(flex:1.2) + 右侧表单(300px)

**拖拽区三状态：**
| 状态 | 外观 | 按钮 |
|------|------|------|
| 未选择 | 虚线边框，引导文案，🎵图标 | 灰色(不可点) |
| 已选择 | 绿色虚线边框，显示文件名+大小 | 霓虹渐变(可点击) |
| 上传中 | 霓虹进度条 + 百分比 | 进度条替代按钮 |

**右侧表单：**
- title 输入框（必填，1-200字符）
- artist 输入框（必填，1-200字符）
- 风格选择：8个圆角标签，单选，选中态霓虹高亮
- 情绪选择：6个圆角标签，单选
- 上传按钮：5条件全满足后启用

**交互：**
- 拖拽高亮：文件拖入时边框亮起霓虹橙色+背景光晕
- 文件可移除：×按钮重新选择
- 支持粘贴文件链接
- 上传成功：✓动画 → 1.5s → 跳转新歌曲播放页
- 上传失败：红色边框 + 错误信息浮动
- 返回按钮：确认弹窗（有未提交数据时）

### 3.4 播放页

**布局：** 延续 india-radio.html 五区域覆盖层

| 区域 | 位置 | 内容 |
|------|------|------|
| 左上图标栏 | top:28px left:28px | 🔊音量 ♡收藏 ⚙设置 (40px圆形玻璃按钮) |
| 右上品牌 | top:30px right:90px | "HEAR THE 🌍" 13px |
| 右侧控制 | right:18px top:50% | □展开 +放大 -缩小 ⛶全屏 (34px垂直排列) |
| 左侧电台面板 | left:28px top:45% | 310px卡片：缩略图(160px天际线+LIVE) + 歌曲信息 + 统计 + 停止按钮 |
| 底部播放器 | bottom:0 | 84px：唱片(54px旋转) + 信息 + 进度条(280px霓虹) + 音量(90px) + 播放按钮(34px) |

**新增：场景主题切换器（点击⚙打开）**
- 320px宽面板从右侧滑入，半透明遮罩背景
- 自动匹配 + 6个手动场景选项，单选圆点
- 画面效果开关：粒子特效 / Bloom / 暗角遮罩

### 3.5 个人主页

**布局：**
- 顶部：头像(80px,可编辑✎) + 昵称 + @用户名 + 统计数据(上传/收藏/歌单) + 编辑资料按钮
- Tab栏：我的歌单 / 上传歌曲 / 播放历史 / 我的收藏
- 底部播放器栏(68px)

**4个Tab内容：**
| Tab | 展示方式 | 特殊处理 |
|-----|---------|----------|
| 我的歌单 | 横向滚动卡片，渐变色封面+歌单名+歌曲数+公开/私密 | 末尾 +新建歌单 虚线卡片 |
| 上传歌曲 | 紧凑列表：小封面(36px)+歌名+艺术家+标签+播放次数 | 无数据时显示空状态插画 |
| 播放历史 | 时间分组(今天/昨天/更早)，右侧显示播放时间 | 点击重新播放 |
| 我的收藏 | 同上布局，额外显示收藏时间，支持取消收藏 | 心形图标点击切换 |

**交互：**
- 头像编辑：点击✎ → 文件选择器 → 1:1裁剪上传
- 编辑资料：Modal弹窗：昵称修改+保存
- 新建歌单：Modal弹窗：名称+描述+公开/私密切换
- 空状态：居中插画 + "还没有..." + 引导按钮

## 5. 过渡动画规范

| 场景 | 动画 | 时长 |
|------|------|------|
| 登录→首页（登录成功） | 表单卡片缩小消失 + 背景变亮 + 首页滑入 | 0.6s |
| 登录⇄注册切换 | 卡片 rotateY(180deg) 翻转 | 0.6s |
| 首页→播放页 | 歌曲卡片放大为全屏 + 3D场景展开 | 0.5s |
| 播放页切歌 | 场景淡入淡出 | 1.2s |
| 设置面板打开 | 从右侧滑入 | 0.3s |
| Modal弹窗 | scale(0.9)→scale(1) + opacity | 0.2s |
| 页面路由切换 | AnimatePresence exit/enter | 0.3s |
