交互细节

  • 拖拽高亮：文件拖入拖拽区时边框亮起霓虹橙色 + 背景光晕微提
  • 文件预览：选择文件后在拖拽区显示：文件名 + 大小 + 格式图标。可点击 × 移除重新选择
  • 按钮启用条件：audio 文件已选 + title 非空 + artist 非空 + style 已选 + mood 已选 → 按钮变亮可点击
  • 上传成功：拖拽区变为 ✓ 动画 + 短暂停留 1.5s → 自动跳转到新歌曲的播放页
  • 上传失败：拖拽区边框变红 + 错误信息浮动显示，3 秒后恢复
  • 返回按钮：左上角 ← 返回上一页（首页或播放页），未提交的数据会丢失（需确认弹窗）
