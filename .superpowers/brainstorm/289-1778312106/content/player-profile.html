个人主页 4 个 Tab 内容区

📋 我的歌单 横向滚动卡片列表，每张卡片：渐变色封面 + 歌单名 + 歌曲数 + 公开/私密标识。末尾 +新建歌单 虚线卡片
🎵 上传歌曲 歌曲列表（与首页网格类似但紧凑），每行：小封面(36px)+歌名+艺术家+标签+播放次数。点击行进入播放
🕐 播放历史 按时间倒序列表，分组显示（今天/昨天/更早），每行右侧显示播放时间。点击重新播放
❤️ 我的收藏 与上传歌曲 Tab 布局一致，额外显示收藏时间。支持取消收藏（心形图标点击切换）

个人主页交互细节

  • 头像编辑：点击右下角 ✎ → 弹出文件选择器 → 上传裁剪（1:1）→ 调用 POST /api/songs/{id}/cover 类似逻辑
  • 编辑资料：点击右上角按钮 → 弹出小型玻璃态 Modal：昵称修改 + 保存
  • 新建歌单：点击 + → 弹出 Modal：歌单名称 + 描述 + 公开/私密切换 → 创建成功添加到列表
  • 返回按钮：左上角 ← 返回首页
  • 空状态：Tab 内无数据时显示居中插画 + "还没有上传歌曲" + 引导按钮
